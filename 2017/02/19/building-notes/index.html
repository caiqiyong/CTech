    关于Github Pages

  Github Pages是Github提供的静态站点托管服务，Github Pages给每个站点分配了1GB的空间用来托管代码，每月享有100GB的软件带宽。博主在本地文件夹中查看，发表的三篇文章带图片资源一起仅仅占用了344KB，因此1GB的配额对于搭建博客写文章还是绰绰有余了。更重要的是Githunb Pages免费！当然在Github pages搭建博客也有不好的地方，比如博客是静态的，不能使用PHP、Python等服务器端代码。但对于广大免费用户来说，应该是喜出望外！ Github Pages官网: https://pages.github.com/

关于Hexo

  Hexo是一个快速、简洁、高效的博客框架。可以快速把Markdown渲染成HTML网页。Markdown官方页面有海量插件和主题可供选择和下载。使像博主这样没有HTML基础的小白搭建独立博客成为可能。Hexo 官网: https://hexo.io/
